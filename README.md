# Beauty Salon Landing Page

A responsive landing page for a beauty salon business, built with HTML, CSS, JavaScript, and PHP.

## Features

- Responsive design using Bootstrap
- Image gallery with Swiper slider
- Services and amenities showcase
- Contact / enquiry form (PHP backend)
- Floor plans and location section
- Thank-you page after form submission

## Tech Stack

- HTML5, CSS3, JavaScript (jQuery)
- Bootstrap
- Swiper.js (sliders)
- Font Awesome (icons)
- PHP (form handling via `index.php` and `ajax/`)

## Project Structure

```
.
├── index.html              # Main landing page
├── index.php               # PHP entry / form handler
├── thank-you.html          # Confirmation page
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── fonts/                  # Web fonts
├── ajax/                   # AJAX libraries (Font Awesome)
└── poem/                   # Additional assets (images, gallery, plans)
```

## Getting Started

Serve the project root with any PHP-capable web server (Apache, XAMPP, etc.) and open `index.php` in a browser.
